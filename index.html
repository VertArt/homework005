1. Универсальный селектор - Универсальные селекторы CSS обозначаются обыкновенной звездочкой «*». Под звездочкой может подразумеваться любой элемент разметки HTML-страницы.
    Пример:
    * {
    font-size: 16px;
    }
  2. Селектор по названию тега - Запись селекторов по тегам также не представляет из себя никакой сложности. Выглядит она следующим образом:
    Пример:
    h1 {
    color: red;
    }
  3. Cелектор по классу - Название класса может быть каким угодно, главное, чтобы оно соответствовало тому элементу, который описывается CSS-правилом.
    Пример:
    .main-header {
    background-color: #99ffcc;
    }
  4. Селектор по id - К атрибутам HTML-тегов относится и id. Атрибут этот применяется к индивидуальному элементу, т.е. элемент с id может быть в единственном экземпляре на странице согласно правилам семантики.
    Пример:
    #heading {
    font-size: 35px;
    }
  5. Селектор по атрибуту - селектор по атрибутам HTML-тегов.
    Пример:
    a[href = "https://loftblog.ru/"] {
    color: black;
    }
  6. Селекторы потомков, или контекстные селекторы - Под CSS селекторами потомков, или контекстными селекторами подразумевается ситуация, когда один элемент HTML-страницы вложен в другой, причем не обязательно он должен быть первым потомком.
    Пример:
    .wraping p {
    padding: 15px;
    }
  7. Селекторы дочерние (только первые) - Дочерние селекторы CSS отличаются от селекторов потомков тем, что под дочерним подразумевается только первый потомок, никакие «внуки» и «правнуки» не попадают в эту категорию.
    Пример:
    .wraping>p {
    `padding: 15px;
    }
  8. Селекторы сестринские, или соседние (расположенные на одном уровне) - При помощи сестринского селектора создается CSS-правило для HTML-элементов, расположенных на одном уровне, причем тот элемент, который записывается как сестринский должен быть самым ближайшим из себе подобных к тому элементу, к которому он записывается в «сестры».
    Пример:
    h1 + p {
    padding-bottom: 30px;
    }
  9. Селекторы псевдоклассов (селекторы состояния) - К селекторам псевдоклассов относятся селекторы CSS, которые описывают состояния элементов при совершении над ними какого-либо действия.
    Пример:
    a:hover {
    text-decoration: none;
    }
  10. Селекторы псевдоэлементов - Их прелесть заключается в том, что с их помощью (::before, ::after) можно добавлять различные элементы на веб-странички, но при этом псевдоэлементы никак не будут отражены в HTML-разметке этих страниц. Записываются они по отношению к элементу через двойное двоеточие.
    Пример:
    p::first-letter{
    color: red;
    }
